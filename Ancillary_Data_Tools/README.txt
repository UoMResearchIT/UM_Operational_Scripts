Scripts for running ANTS preprocessors.

See documentation at:
https://cms.ncas.ac.uk/miscellaneous/ants-container/
and:
https://code.metoffice.gov.uk/doc/ancil/ants/latest/introduction.html

These scripts are written for the ANTS v0.18 singularity container.
However, where necessary, updated scripts have been included in 
this repository.

To use these scripts you will need to download these files:
ESACCI-LC-L4-LCCS-Map-300m-P1Y-2010-v2.0.7b.nc
modis_landcover_class_qd.asc
